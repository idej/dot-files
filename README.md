Dot-file
=========
Configuration files, fonts and even wallpaper for my mac.

Usage
=========
1. Install Homebrew

`/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"`

2. Install git

`brew install git`

3. Clone this repo

`git clone https://github.com/idej/dotfiles.git`

4. Run install

`./install.sh`
